# Library
A virtual library that stores books. This is a project from [The Odin Project](<https://www.theodinproject.com/>).

## Features
* Add book to library
* Remove book from library
* Render books in html

* HTML
* CSS
* JavaScript
* Figma
* Visual Studio Code
